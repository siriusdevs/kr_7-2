1. Создаем удаленный репозиторий:
Нам нужно создать пустой удаленный репозиторий на GitHub

2. Далее необходимо получить URL удаленного репозитория(ssh или http)

3. Добавьте удаленный репозиторий в локальный репозиторий:
В вашем локальном репозитории выполните команду git remote add, указав URL удаленного репозитория:

git remote add origin https://github.com/.git


4. Далее отправим код на удаленный репозиторий:

git push -u origin master


Эта команда отправит наш код на удаленный репозиторий.

Теперь наш локальный репозиторий связан с удаленным и мы можем пользоваться всеми командами
